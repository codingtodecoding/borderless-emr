=====================================

ACCESS LEVELS
=============

Public Users:
- Can visit homepage
- Can login
- Can register
- Cannot access admin panel

Registered Users (user role):
- Can login
- Can access homepage
- CANNOT access admin panel
- CANNOT manage users/roles

Admin Users (admin role):
- Can login
- Can access admin dashboard
- Can manage all users
- Can manage roles
- Can view activity logs
- Full access to admin panel
